User Story：智慧捷運乘車輔助系統

背景故事

小明是一名剛到台北讀書的大學生。

某天，他要前往台北科技大學參加活動。他知道要在忠孝新生站下車，並從4號出口離開，但他發現捷運站很大，下車後常常需要走很長一段路才能找到正確出口。

此外，在轉乘其他路線時，他也不清楚應該站在哪一節車廂，才能最快抵達手扶梯、電梯或轉乘通道。

因此，小明希望有一套系統能夠根據他選擇的車站與出口，提供最適合的乘車位置與站內資訊，減少尋找出口及轉乘的時間。



User Story 1：出口對應車廂推薦

As a 台北科大學生
I want to 選擇我要前往的車站出口
So that 我可以知道應該搭乘哪一節車廂及哪個車門，最快到達出口，讓我可以趕上早八的課。

驗收條件
使用者可選擇車站與出口。
系統顯示建議車廂位置。
系統顯示距離最近的手扶梯或電梯。


User Story 2：轉乘資訊查詢

As a 捷運乘客
I want to 查詢不同路線之間的轉乘資訊
So that 我可以更快速及順利的完成轉乘。

驗收條件
系統顯示轉乘路線。
系統顯示預估步行時間。
系統顯示建議車廂位置。
系統顯示最近的手扶梯或電梯位置。


User Story 3：站內設施查詢

As a 捷運乘客
I want to 查詢車站內的設施位置
So that 我可以快速找到需要的服務。

驗收條件
可查詢廁所位置。
可查詢置物櫃位置。
可查詢飲水機位置。
可查詢服務台位置。
顯示接近的出口資訊。


User Story 4：無障礙資訊查詢

As a 行動不便乘客
I want to 查詢電梯及無障礙設施位置
So that 我能夠更方便地搭乘捷運。

驗收條件
顯示電梯位置。
顯示建議搭乘車廂。
顯示最近出口資訊。


系統價值

本系統不負責規劃捷運路線，而是專注於提供捷運站內導航與乘車輔助資訊。

透過整合出口、車廂、轉乘及站內設施資料，幫助乘客更快速找到正確出口、縮短轉乘時間，並提升整體搭乘體驗。
